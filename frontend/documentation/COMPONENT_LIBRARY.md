# Momentum Science Academy - Component Library

## Reusable Components

### Layout Components

#### Navbar
- Logo with academy name
- Navigation links (responsive)
- Auth buttons (Student/Teacher login)
- Mobile menu toggle
- Sticky positioning

#### Footer
- Logo and description
- Quick links
- Contact info
- Social media links
- Newsletter subscription

### Card Components

#### ProgramCard
- Program name and duration
- Subjects covered with badges
- Highlights list
- Fee display
- Enroll button

#### FacultyCard
- Faculty photo/avatar
- Name and subject
- Experience and qualifications
- Specialization
- Toppers count

#### ResourceCard
- Resource title
- Type badge (PYQ/Notes/Assignment)
- Download button
- Subject and class tags
- Last updated date

#### BlogCard
- Blog title and excerpt
- Category badge
- Publication date
- Author name
- Read more link

### Form Components

#### InquiryForm
- Name, email, phone fields
- Program selection dropdown
- Submit button
- Success/error messages

#### LoginForm
- Email input
- Password input
- Remember me checkbox
- Forgot password link
- Login/Register toggle
- OAuth option

#### ResourceUploadForm
- Title input
- Description textarea
- Resource type selector
- Subject and class dropdowns
- File upload area
- Publish checkbox

### Dashboard Components

#### StatCard
- Icon
- Stat value
- Stat label
- Trend indicator (optional)

#### ChartWidget
- Title
- Line/Bar chart
- Legend
- Responsive sizing

#### DataTable
- Sortable columns
- Pagination
- Search/filter
- Row actions (edit, delete)
- Mobile-friendly

### Filter Components

#### ResourceFilter
- Class selector
- Exam type selector
- Subject filter
- Resource type toggle
- Date range picker

### Modal Components

#### ConfirmDialog
- Title and message
- Action buttons
- Icon indicator

#### FormModal
- Header with close button
- Form content
- Action buttons
- Overlay background

### UI Primitives (shadcn)
- Button (variants: primary, secondary, outline, ghost)
- Card (default, hover effects)
- Input
- Select
- Textarea
- Badge
- Alert
- Dialog
- Tabs
- Accordion
- Skeleton (for loading states)
